### Security & System Tools

- **[PortKill](https://github.com/mr-tanta/portkill)** — Zero-dependency (~72KB) port management and network analysis tool for macOS/Linux. Kill processes, manage Docker containers, benchmark connections, and run remote ops via SSH — all in pure Bash. Install: `brew install mr-tanta/portkill/portkill`

- **[Securing Production Servers](https://github.com/mr-tanta/securing-production-servers-with-best-practices)** — Modern Linux hardening reference for cloud-native environments. Covers nftables, container hardening, IaC scanning, zero-trust, and SIEM integration for engineers working with CI/CD and automated deployments.

### Compliance & Integration

- **[NDPR Toolkit](https://github.com/mr-tanta/ndpr-toolkit)** — Production-ready React component library for Nigeria Data Protection Act (NDPA 2023) compliance. 8 modules covering consent management, data subject rights, DPIA, breach notification, and more. Published on npm: `pnpm add @tantainnovative/ndpr-toolkit`

- **[Intercom Next.js 14 Guide](https://github.com/mr-tanta/intercom-next-js-14-guide)** — Step-by-step guide and working codebase for integrating Intercom chat with Next.js 14 App Router. Environment config, component architecture, and TypeScript types included.
